## Study Guide Format

Each module includes:
- **Duration**: Estimated time for the module
- **Topics**: Number of topics covered
- **Level**: Beginner or Intermediate
- **Overview**: Summary of module content
- **Learning Objectives**: Key competencies to develop
- **Topic Breakdowns**:
  - Overview of each topic
  - Key Concepts: Important terms and definitions
  - How It Works: Process descriptions
  - Takeaways: Key points to remember
- **Module Summary**: Comprehensive recap tying concepts together
- **CSA Exam Tips**: 5-6 specific points relevant to CSA examination
